## Notes

1. **Vimeo Videos**: Use Vimeo ID in the frontend, or embed URL format: `https://player.vimeo.com/video/{VIDEO_ID}`
2. **Local Videos**: Use direct file paths starting with `/portfolio/`
3. **Thumbnails**: Always use the WebP thumbnails when available for better performance
4. **Aspect Ratios**: 
   - 9:16 (Stories) - Most Vimeo videos
   - 1:1 (Square) - Some product videos
   - 16:9 (Landscape) - Website videos
   - 4:3 (Standard) - Comet AI Ad
